# AI-HED Brand Guidelines

## Overview

This skill provides the complete brand specifications for the AI-HED project (Artificial Intelligence in Higher Education Teaching and Learning), an Erasmus+ KA2 Strategic Partnership co-funded by the European Commission. Use these guidelines for all project deliverables, presentations, documents, and digital materials.

## Brand Identity Quick Reference

### Core Colours
- **Primary Green**: `#628c68` (RGB: 98, 140, 104) — main brand identity, solid colour applications
- **White**: `#ffffff` — text on dark/green backgrounds, clean contrast
- **Black**: `#000000` — text on light backgrounds, logo text

### Gradient (Icon and Logo)
The icon and logo use a green gradient flowing from top-left to bottom-right:
- Light end: `#63a376` (RGB: 99, 163, 118)
- Mid tones: `#4e835e`, `#4a7c5a`, `#467655`
- Dark end: `#2c4d36` (RGB: 44, 77, 54)

### Typography
- **Headings**: Franklin Gothic Medium
- **Body text**: Helvetica (fallback: Verdana, then Arial)
- **Logo name**: Franklin Gothic (thin/light weight)
- **Logo tagline**: Helvetica Light

### Heading Sizes
| Level | Font | Size (px / pt) |
|-------|------|-----------------|
| H1 | Franklin Gothic Medium | 48px / 36pt |
| H2 | Franklin Gothic Medium | 40px / 30pt |
| H3 | Franklin Gothic Medium | 32px / 24pt |
| H4 | Franklin Gothic Medium | 28px / 21pt |
| H5 | Franklin Gothic Medium | 24px / 18pt |
| H6 | Franklin Gothic Medium | 20px / 15pt |
| Body | Helvetica | 16px / 12pt |

### The Icon
An abstract neural/brain connection symbol representing the connection of neurons that happens when we learn. Never use the icon in isolation — it must always appear alongside the project name or logo.

### The Logo
Two versions exist:
- **Full**: Icon + "AI-HED" + tagline "Artificial Intelligence in Higher Education Teaching and Learning"
- **Short-hand**: Icon + "AI-HED" (no tagline)

Each version has four variants:
1. Solid green on white/light background
2. Gradient green on white/light background
3. White (inverted) on dark background
4. White (inverted) short-hand on dark background

## Application Guidelines

### Presentations
- Use white backgrounds with green accents for content slides
- Dark green or gradient backgrounds for title/impact slides with white text
- Franklin Gothic Medium for all headings
- Helvetica at 18pt for body text on slides (increased from 12pt for readability)
- Maintain generous whitespace

### Documents
- Franklin Gothic Medium for headings at specified sizes
- Helvetica 12pt for body text
- Use bold for key points, italic for emphasis/foreign words
- Blockquote styling for quoted text
- Maintain clear hierarchy between heading levels

### Digital/Web
- Primary green `#628c68` for headers and key sections
- Gradient green for decorative elements and the icon
- White backgrounds with green accents for readability
- Helvetica or system sans-serif for body text

## Design Principles

**Style Keywords**: Academic, professional, clean, accessible, modern, institutional

**Core Aesthetic**:
- Clean layouts with clear hierarchy
- Accessibility-first (font choices prioritise dyslexia-friendly reading)
- Institutional but modern feel
- Green colour palette connects to growth and learning
- Generous whitespace
